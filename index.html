Digital Pressure Monitoring Cluster (V1 - MVP)

Designed and developed a real-time embedded system for Indian Railways to monitor pressure sensors and display gauge values with high responsiveness.

Key Highlights

  • STM32-based system with single-loop architecture (no RTOS)
  • Loop cycle: 135 ms ensuring real-time updates
  • Supports 6 CAN-based pressure sensors
  • Pressure range up to 11 bar
  • Data logging every 3 seconds to SD card

Interfaces Used

  • CAN → Pressure sensors
  • UART → DWIN display
  • SPI → SD card
  • I2C → RTC

Features

  • Real-time pressure gauge visualization
  • Warning, error, and status messages
  • USB firmware update with fallback mechanism
  • USB data dump from SD card
  • Multi-function control using only 4 buttons

Your Contribution

  • Designed complete firmware architecture
  • Implemented CAN communication for 6 sensors
  • Developed multi-function button logic (single-button sequences)
  • Handled SD logging and USB data transfer
  • Optimized loop timing to achieve 135 ms response
