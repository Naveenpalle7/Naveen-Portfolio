Digital Pressure Monitoring Cluster — Indian Railways (V1 MVP)

Designed and developed a real-time pressure monitoring cluster using STM32, capable of handling multiple sensors over CAN and displaying live data with minimal latency.

System Architecture

  • STM32 MCU (Main Controller)
  • 6 Pressure Sensors via CAN
  • DWIN Display via UART
  • SD Card via SPI
  • RTC via I2C
  • USB for firmware update & data dump

Engineering Highlights

  • ~135 ms loop cycle without RTOS
  • Handled CAN, UART, SPI, I2C in single loop
  • Multi-function button logic (4 buttons)
  • Data logging every 3 seconds

Challenges & Solutions

Multi-interface without RTOS: Designed deterministic master loop

Limited UI: Implemented sequence-based button control

Firmware safety: Added fallback mechanism

Impact

  • Real-time railway monitoring
  • Improved fault visibility
  • Reduced maintenance effort
